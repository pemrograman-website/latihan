Alert Tanpa Tautan

Elemen Alert dengan class="close" dan data-dismiss="alert" digunakan untuk menutup kotak alert.

Kelas .alert-dismissible menambahkan padding ekstra pada tombol tutup.

Success! This alert box could indicate a successful or positive action.
Info! This alert box could indicate a neutral informative change or action.
Warning! This alert box could indicate a warning that might need attention.
Danger! This alert box could indicate a dangerous or potentially negative action.

Alert Dengan Tautan

Tambahkan kelas .alert-link pada tautan di dalam kotak alert untuk membuat tautan yang sesuai.

Success! You should read this message.
Info! You should read this message.
Warning! You should read this message.
Danger! You should read this message.
